This plan CREATES the foundational types that all subsequent phases depend on.
The executor must implement these exact contracts from README.MD:

Consumer → Provider JSON schemas:
- SearchRequest: action="search", destination, check_in, check_out, guests, hard_constraints, weights, context
- BookRequest: action="book", option_id, guest_name, check_in, check_out
- CancelRequest: action="cancel", booking_id
- ReadRequest: action="read", booking_id

Provider → Consumer JSON schemas:
- SearchResponse: action="search_results", options[], negotiation_note
- HotelOption: id, name, price_per_night, rating, distance_km, free_cancellation, amenities[], total_score, score_breakdown, tag

Weights must have these 5 fields (0–1 each, sum to 1.0):
- price, location, rating, cancellation, amenities

Scoring formula (from README):
- price_score = 1 - (price / max_budget)
- location_score = 1 - (distance_km / max_distance)  # max_distance = 5.0km
- rating_score = rating / 5.0
- cancellation_score = 1.0 if free, 0.0 if not
- amenities_score = matched_count / desired_count
- total_score = weighted sum of all 5 dimension scores
</interfaces>

<task type="auto">
  <name>Task 2: Create defensive scoring engine in provider_agent/scorer.py</name>
  <files>provider_agent/__init__.py, provider_agent/scorer.py</files>
  <action>
Create `provider_agent/__init__.py` (empty or with exports).

Create `provider_agent/scorer.py` with defensive scoring implementation:

1. **Constants** at top of file:
   ```python
   DEFAULT_MAX_BUDGET = 500.0  # EUR
   DEFAULT_MAX_DISTANCE = 5.0  # km
   DEFAULT_RATING = 3.0  # 3-star if missing
   ```

2. **clamp(value: float) -> float** helper:
   - Returns `max(0.0, min(1.0, value))` to ensure 0–1 range

3. **calculate_price_score(price: float, max_budget: float) -> float**:
   - If max_budget <= 0: use DEFAULT_MAX_BUDGET
   - If price <= 0: return 0.5 (unknown = neutral)
   - If price >= max_budget: return 0.0 (at/over budget = worst)
   - Formula: `clamp(1 - (price / max_budget))`

4. **calculate_location_score(distance_km: float, max_distance: float = DEFAULT_MAX_DISTANCE) -> float**:
   - If distance_km < 0: return 0.5 (unknown = neutral)
   - If distance_km >= max_distance: return 0.0 (too far)
   - Formula: `clamp(1 - (distance_km / max_distance))`

5. **calculate_rating_score(rating: float | None) -> float**:
   - If rating is None or <= 0: use DEFAULT_RATING
   - Formula: `clamp(rating / 5.0)`

6. **calculate_cancellation_score(free_cancellation: bool) -> float**:
   - Return 1.0 if True, 0.0 if False

7. **calculate_amenities_score(hotel_amenities: list[str], desired_amenities: list[str]) -> float**:
   - If desired_amenities is empty: return 1.0 (no requirements = full score)
   - Calculate: matched = len(set(hotel_amenities) & set(desired_amenities))
   - Return: matched / len(desired_amenities)

8. **score_hotel(hotel: dict, weights: Weights, max_budget: float, desired_amenities: list[str] | None = None) -> tuple[float, ScoreBreakdown]**:
   - Calculate all 5 dimension scores using the functions above
   - Create ScoreBreakdown with the 5 scores
   - Calculate total: weighted sum of (dimension_score * weight) for all 5 dimensions
   - Return (clamp(total), breakdown)

9. **score_and_rank(hotels: list[dict], weights: Weights, max_budget: float, desired_amenities: list[str] | None = None) -> list[HotelOption]**:
   - Score each hotel using score_hotel
   - Create HotelOption instances with scores
   - Sort by total_score descending
   - Return top 3 results with tags assigned:
     - BEST_BALANCE: highest total_score (always #1 after sort)
     - CHEAPEST: lowest price_per_night among top 3 (if different from BEST_BALANCE)
     - HIGHEST_RATED: highest rating among top 3 (if different from BEST_BALANCE and CHEAPEST)

Import from shared.models: `from shared.models import Weights, ScoreBreakdown, HotelOption`
  </action>
