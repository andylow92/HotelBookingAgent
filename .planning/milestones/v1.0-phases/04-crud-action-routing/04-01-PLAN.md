  <name>Task 1: Create action router with dispatch dict and all CRUD handlers</name>
  <files>provider_agent/agent.py</files>
  <action>
Create `provider_agent/agent.py` with these components:

**Entry point — `handle_action(payload: dict) -> dict`:**
- Extract `action` field from payload
- Look up handler in `_ACTION_HANDLERS` dispatch dict mapping action strings to handler functions
- Return `{"status": "error", "message": f"Unknown action: {action}"}` for unknown actions
- Wrap handler call in try/except: catch `ValidationError` → error response with message, catch `Exception` → error response with `f"Internal error: {e}"`

**Module-level state (locked decisions — resets on restart):**
- `_bookings: dict[str, dict] = {}` — booking store keyed by confirmation ID
- `_last_search_options: dict[str, HotelOption] = {}` — cache search results for book handler lookup
- `_last_raw_hotels: list[dict] = []` — cache raw hotels for potential rescore

**`_generate_booking_id() -> str`:**
- Return `f"BK-{uuid.uuid4().hex[:6].upper()}"` — readable demo-friendly IDs

**`_handle_search(payload: dict) -> dict`:**
- Validate with `SearchRequest(**payload)`
- Call `search(request)` from `provider_agent.search` — returns `(SearchResponse, raw_hotels)`
- Clear and populate `_last_search_options` — key by `opt.id` for each option in response
- Clear and populate `_last_raw_hotels` with raw hotels
- Return `response.model_dump(mode="json")`

**`_handle_book(payload: dict) -> dict`:**
- Validate with `BookRequest(**payload)`
- Look up `request.option_id` in `_last_search_options`
- If not found → return error: `f"Unknown option_id: {request.option_id}"`
- Generate booking ID with `_generate_booking_id()`
- Store full booking record in `_bookings[booking_id]` with fields: `booking_id`, `hotel_name` (from option.name), `hotel_id` (from option.id), `price_per_night` (from option.price_per_night), `check_in` (str), `check_out` (str), `guest_name`, `free_cancellation` (from option.free_cancellation), `status: "confirmed"`
- Return `{"status": "success", "booking_id": booking_id, "hotel_name": option.name}`

**`_handle_read(payload: dict) -> dict`:**
- Validate with `ReadRequest(**payload)`
- Look up `request.booking_id` in `_bookings`
- If not found → return error: `f"Booking not found: {request.booking_id}"`
- Return `{"status": "success", "booking": booking}` (wrap in "booking" key to avoid collision with booking's own "status" field)

**`_handle_cancel(payload: dict) -> dict`:**
- Validate with `CancelRequest(**payload)`
- Look up `request.booking_id` in `_bookings`
- If not found → return error: `f"Booking not found: {request.booking_id}"`
- Set `booking["status"] = "cancelled"`
- Derive refund: `"full_refund"` if `booking.get("free_cancellation")` else `"no_refund"` (locked decision: refund depends on free_cancellation field)
- Return `{"status": "success", "booking_id": request.booking_id, "refund_status": refund}`

**Dispatch dict:**
```python
_ACTION_HANDLERS = {
    "search": _handle_search,
    "book": _handle_book,
    "read": _handle_read,
    "cancel": _handle_cancel,
}
```
